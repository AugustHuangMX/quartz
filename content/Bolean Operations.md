Remember the `False and "anything"` would be `False`

## Exercise

```python
a = True
b = “True”
c = a != b

print(c)
print(type(c))

```

Since the  type of `a` is Bool, and the type of `b` is [[String]]. So `a != b` is true.

So `c` is true. Also, the type of `c` would be Bool, too.
